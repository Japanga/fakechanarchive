Description

FakeChan Archive is a manually operated archive of threads on the FakeChan site that were visited and used by guests. Only "guest party board snapshots" are archived, and ones with high amounts of users. Your uploads to FakeChan will likely remain there for a long while, but eventually have to be removed if we get too close to the storage limits. Moderation on FakeChan is very lax, but I will instantly delete any unlikable images from the feed if they are uploaded. FakeChan does not preserve moderation acts or deletes, your un-approved material will be quietly and swiftly deleted from the uploads feed in the Cloudinary dashboard. If you type $DESU$ on your post in the archive comments, it will turn into an animated glowing effect. We were unable to add $DESU$ texting to the main FakeChan site because it conflicted with the code for the Cloudinary image uploading, so you will only be able to use the $DESU$ features as a special treat extra on the Archive site. This gives the Archive an element that makes it unique and special compared to the base site.
